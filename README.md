# Personal Research Page Starter

This repository is a cleaned personal website starter based on:
- https://github.com/jonbarron/jonbarron.github.io

## Edit these files
- `index.html`: your profile, links, publications, news
- `stylesheet.css`: styling

## Notes
- Replace all bracketed placeholders in `index.html`.
- Add your own images/files under `images/` as needed.

## Deploy on GitHub Pages
1. Create a GitHub repository and push this folder to the `main` branch.
2. In GitHub, open `Settings -> Pages`.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Push changes to `main`; the workflow in `.github/workflows/deploy-pages.yml` will deploy automatically.
5. Your site URL will be:
   - `https://<username>.github.io/` for a user/organization site repo named `<username>.github.io`
   - `https://<username>.github.io/<repo-name>/` for a project site repo
